ЗАДАЧА

Представьте, что перед Вами коридор, вдоль которого стоят 100 запертых шкафчиков.
Сначала Вы открываете их все, потом закрываете каждый второй. После этого подходите к
каждому третьему шкафчику и открываете его, если он закрыт, и закрываете, если открыт.
Назовём эту процедуру переключением шкафчика. На шаге n переключить нужно шкафчик 
с номером n. На сотом шаге переключенным окажется только шкафчик с номером 100.
Сколько шкафчиков окажутся открытыми после сотого шага?


РЕШЕНИЕ

Для решения этой задачи был написан метод int countingOfOpenLockers(int countOfLockers), 
на вход которого подаётся количество шкафчиков, а возвращённым значением является
количество открытых шкафчиков.
